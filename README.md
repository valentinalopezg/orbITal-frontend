## ⚙️ Instalación

```bash
git clone https://github.com/valentinalopezg/orbITal-frontend.git
cd orbITal-frontend
npm install
```

## ▶️ Correr el proyecto

```bash
ng serve
```

Abrir en: **http://localhost:4200**
